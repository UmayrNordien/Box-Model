What is a CSS Box-Model

D e f i n i t i o n :

We refer to the Box-Model as the design and layout. Its essentially a container that embodies the HTML element you want like a wrapper


  • M a r g i n s :

    • The Margin property is the area OUTSIDE the border


  • B o r d e r s :

    • Covers the padding and the content, it sets the STYLE, COLOR and WIDTH of the border


  • P a d d i n g :

    • Creates space around the CONTENT

  • C o n t e n t :

    • This property displays the text and images, mostly what the viewer of the webpage sees
